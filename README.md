# ⌚ The World of Watches (WoW)

**The World of Watches (WoW)** is a premium, robust, and feature-rich Django e-commerce platform built for watch enthusiasts and online retailers. It delivers a modern, fluid shopping experience—from product discovery and category filtering to secure cart management and dual-payment checkout systems.

Styled with a responsive and sleek design, the application leverages **Django** for secure, role-based backend architecture, offering a high-performance experience for customers and shop administrators alike.

---

## 🚀 Key Features

- **Dynamic Search & Category Filtering** – Search watches instantly by name, description, or filter them by categories (Analog, Digital, Sports).
- **Full-featured Shopping Cart** – Dynamic cart management to add products, adjust quantities with auto-recalculated pricing, and remove items seamlessly.
- **Dual Checkout System** – Choose between Cash on Delivery (CoD) or Online Wallet (JazzCash, EasyPaisa, etc.). Online payments support screenshot uploads for admin verification.
- **Auto-Logout Security** – Integrated security middleware that automatically logs out idle users after 30 minutes of inactivity.
- **Google OAuth Integration** – Pre-configured social login capabilities using `django-allauth` and `social-auth-app-django`.
- **Custom User Profiles** – Personalized profile dashboards to update info (email, phone, first/last name) and verify password changes.
- **Order Management & History** – Real-time order tracking for customers to view order dates, items, status, and payment screenshots.

---

### 🔐 Role-Based Access
- **Customers** &rarr; Search watches, add items to cart, manage quantities, place orders (Cash on Delivery or upload Online Payment receipts), track order status, and update profile details.
- **Admins** &rarr; Add and update watches, configure payment wallets, process/ship orders, verify manual transaction screenshots, and manage user accounts.

### 👤 Secure Account Management
- **Profile Updates:** Email, Phone, First Name, Last Name.
- **Security:** Standard and Google OAuth authentication, session cookie security, and password reset/change utilities.

## 🔧 Getting Started

### Prerequisites
- Python 3.8+
- Git

### 1. Clone the repository:
```bash
git clone https://github.com/NoorMustafa4556/World-Of-Watches-Django.git
```
### 2. Navigate To Directory
```bash
cd World-Of-Watches-Django
```

### 3. Create and activate a virtual environment:
```bash
python -m venv venv
```
### (i) Windows
```bash
.\venv\Scripts\activate
```

### (ii) Mac/Linux
```bash
source venv/bin/activate
```

### 4. Install dependencies:
```bash
pip install -r requirements.txt
```

### 5. Apply migrations:
```bash
python manage.py makemigrations
```
```bash
python manage.py migrate
```

### 6. Create Superuser (Admin account):
```bash
python manage.py createsuperuser
```

### 7. Run the server:
```bash
python manage.py runserver
```
